# Talent Retention & Compensation Audit Dashboard

A strategic HR analytics dashboard designed to analyze workforce stability, pay equity, and performance metrics. This report provides actionable insights to help organizations optimize talent retention strategies and understand the drivers behind employee attrition.

## 👥 Workforce Overview

The dashboard monitors key performance and stability indicators across the organization:

* **Total Employees:** 30
* **Attrition Rate:** 13.33%
* **Average Performance Score:** 3.80 / 5.0
* **Average Salary:** $77.43K

## 📊 Core Visualizations

1.  **Average Salary by Department:** A continuous area chart mapping compensation levels across Engineering, Finance, Human Resources, IT Support, and Sales.
2.  **Attrition by Department:** A horizontal bar analysis identifying which departments experience the highest turnover, with **Human Resources** showing a significant lead in attrition volume.
3.  **Attrition by Performance & Reason:** A detailed breakdown of why employees leave (Terminated vs. Resigned for Better Opportunities, Relocation, or Personal Reasons) and how those departures correlate with performance.
4.  **Average Salary by Performance:** A scatter plot (Quadrant Analysis) exploring the relationship between compensation and employee performance scores, identifying high-performers' pay positioning.

## 🤖 Data Generation
The data used to build this dashboard is **fully synthetic and AI-generated**.
* **Methodology:** Generated using a custom Python script to simulate HR information system (HRIS) data.
* **Purpose:** Created for portfolio demonstration to showcase expertise in HR analytics, data modeling, and dashboard design while ensuring complete data privacy.

## 🛠️ Tech Stack
* **Visualization Tool:** Power BI
* **Data Manipulation:** PostgreSQL
* **Design Theme:** Professional Dark Mode with Cyber-Cyan accents for high readability.
